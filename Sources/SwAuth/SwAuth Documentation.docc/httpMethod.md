# ``SwAuth/HTTPRequest/httpMethod``

## Overview

Set the HTTP method of the ``HTTPRequest``:

```swift
httpRequestInstance.httpMethod = .POST
```

Other possible values are as follows:

- `.GET`
- `.PUT`
- `.ACL`
- `.HEAD`
- `.POST`
- `.COPY`
- `.LOCK`
- `.MOVE`
- `.BIND`
- `.LINK`
- `.PATCH`
- `.TRACE`
- `.MKCOL`
- `.MERGE`
- `.PURGE`
- `.NOTIFY`
- `.SEARCH`
- `.UNLOCK`
- `.REBIND`
- `.UNBIND`
- `.REPORT`
- `.DELETE`
- `.UNLINK`
- `.CONNECT`
- `.MSEARCH`
- `.OPTIONS`
- `.PROPFIND`
- `.CHECKOUT`
- `.PROPPATCH`
- `.SUBSCRIBE`
- `.MKCALENDAR`
- `.MKACTIVITY`
- `.UNSUBSCRIBE`
- `.SOURCE`
